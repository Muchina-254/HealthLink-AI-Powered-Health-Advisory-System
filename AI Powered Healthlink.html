About HealthLink AI

HealthLink AI is a lightweight machine-learning-driven triage system designed for rural healthcare environments. Community health volunteers collect basic patient symptoms and vitals through a mobile app. The system’s backend runs a trained logistic regression model to classify urgency levels (GREEN, YELLOW, RED), generating instant recommendations and referrals to local clinics.

Core Features

AI-driven Triage

Uses machine learning to assess symptoms and suggest severity categories instantly.

Mobile App for CHWs

Flutter-based app for offline-first patient registration, data collection, and syncing.

Clinic Dashboard

Web dashboard for clinics to view incoming referrals and patient triage statuses.
